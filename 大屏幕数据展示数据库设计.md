#### 大屏数据展示

**大屏展示配置：**

配置编号（主键 INT （DP001、DP002...）  自增，非空)

展示模块（varchar（50） （能源总览 / 光伏总览 / 配电网运行状态 / 告警统计） 非空）、

数据刷新频率（INT 非空（分钟））、

展示字段（TEXT （总能耗 / 光伏发电量 / 高等级告警数）非空）、

排序规则（varchar（50） （按时间降序 / 按能耗降序...）可空）、

权限等级（varchar（50） （管理员 / 能源管理员 / 运维人员）非空）



**实时汇总数据：**

汇总编号（INT 主键 （001..002）自增）、

统计时间（DATETIME ）、

总用电量（DECIMAL(18,2) kWh）、

总用水量（DECIMAL(18,2) m³）、

总蒸汽消耗量（DECIMAL(18,2) t）、

总天然气消耗量（DECIMAL(18,2) m³）、

光伏总发电量（DECIMAL(18,2) kWh）、

光伏自用电量（DECIMAL(18,2) kWh）、

总告警次数（INT）、

高等级告警数（INT）、

中等级告警数（INT）、

低等级告警数（INT）。



**历史趋势数据：**

趋势编号（INT 主键 自增 （001、002...） 非空）、

能源类型（VARCHAR(20) 电 / 水 / 蒸汽 / 天然气 / 光伏 非空）、

统计周期（VARCHAR(20) 日 / 周 / 月 非空）、

统计时间（DATE 非空）、

能耗 / 发电量数值（DECIMAL(18,2)  非空）、

同比增长率（DECIMAL(8,2) % 可空）、

环比增长率（DECIMAL(8,2) % 可空）、

行业均值（VARCHAR(20) 可空）。
