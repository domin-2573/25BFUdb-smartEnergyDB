## 使用方法

1. 执行 `sql/05_测试数据_修正版.sql` 插入测试数据
2. 访问登录页面：`http://localhost:2008/SmartEnergy/login`
3. 使用任意用户ID（如 `001`）和密码 `123456` 登录

## 注意事项

- 数据库中存储的是SHA-256哈希值，不是明文密码
- 登录时输入的是原始密码 `123456`，系统会自动进行SHA-256加密后与数据库中的哈希值比较
- 如果数据库中的密码不是哈希值，需要先执行更新SQL脚本

## 更新现有数据库中的密码

如果数据库中已有用户数据，但密码是占位符（如 `pwdhash001`），可以使用以下SQL更新：

```sql
UPDATE `用户表` 
SET `密码` = '8d969eef6ecad3c29a3a629280e686cf0c3f5d5a86aff3ca12020c923adc6c92' 
WHERE `密码` LIKE 'pwdhash%';
```
